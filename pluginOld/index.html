To make Pali text clickable:
1. it should be marked with class="pli-lang" e.g.: <span class="pli-lang">Pali text </span>

2. styles should be applied in the Head section: <link rel="stylesheet" href="assets/css/paliLookup.css">

3. javascript should be applied after the Pali text, e.g. in the end of the Body section: <script src="assets/js/paliLookup.js"></script>
