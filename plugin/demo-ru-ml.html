Вкл/выкл всплывающее окно с DPD iframe.
Idaṁ kho pana, bhikkhave, dukkhaṁ ariyasaccaṁ —
Здесь будет Ваш любимый перевод. Кликните по тексту Пали, чтобы открыть DPD iframe.

jātipi dukkhā, jarāpi dukkhā, byādhipi dukkho, maraṇampi dukkhaṁ, appiyehi sampayogo dukkho, piyehi vippayogo dukkho, yampicchaṁ na labhati tampi dukkhaṁ — saṅkhittena pañcupādānakkhandhā dukkhā.
Чтобы Пали текст был кликабельным:
1. он должен быть отмечен классом pli-lang, к примеру: <span class="pli-lang">Пали текст</span>

2. стили должны быть подключены в разделе Head: <link rel="stylesheet" href="assets/css/paliLookup.css">

3. javascript должен быть подключен после текста пали, например, в конце раздела Body: <script src="assets/js/paliLookup.js"></script>

4. Скрипт автоматически будет открывать Русскую версию DPD онлайн, если: есть запись в LocalStorage "siteLanguage":"ru" или url содержит /ru/ или url содержит ml.html
