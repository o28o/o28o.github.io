Privacy Policy

Covered Products

Telegram Services

  • Telegram Bots:
    • @Dhammagift_bot - Main bot for Pali dictionary and sutta search
    • @Dgift_bot - Shortname version of the bot
  • Telegram Mini Apps:
    • Search Mini-App - For sutta search within Telegram
    • Reader Mini-App - For reading Pali texts
    • Dictionary Mini-App - For dictionary lookups

PWA & TWA Applications

  • Dhamma.gift Web App – Installable Progressive Web App (PWA) with Shortcuts and Sharing
  • Dhamma.Gift Android TWA App: Available via Google Play
  • Dict.Dhamma.gift Web App – Installable Progressive Web App (PWA) with Shortcuts and Sharing
  • Dict.Dhamma.Gift Android TWA App: Available via Google Play

Browser Extensions

  • Chrome Extension
  • Firefox Add-on
  • Microsoft Edge Add-on

Data Processing

These products process only the minimal technical data required for their operation:

  • For dictionary lookups: Only the clicked/selected word is transmitted to our dictionary services
  • For sutta search: Only search queries and text references are transmitted
  • Technical data: Basic request metadata (IP address, user agent) that web servers normally collect

What We Do Not Collect

  • No personal user information
  • No Telegram user IDs or chat histories
  • No browsing history outside our services
  • No persistent tracking identifiers

Data Transmission

All data is transmitted only to our controlled services:

  • dhamma.gift - For text search and reading functionality
  • dict.dhamma.gift - For dictionary lookups
  • dpdict.net - For comprehensive Pali dictionary results

No third-party services receive any user data from our products.

Data Retention

We do not store:

  • Search queries or lookup words after processing
  • Any user-specific usage patterns
  • Telegram interaction history

Server logs are kept for 30 days maximum for operational purposes only.
