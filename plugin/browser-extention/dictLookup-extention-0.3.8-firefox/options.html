Dictionary Lookup Settings

To enable or edit hotkeys, go to Chrome extentions keyboard shortcuts: chrome://extensions/shortcuts

Enter the base URL for your dictionary lookup (include ?q= at the end)

for English URL is (default)
https://dict.dhamma.gift/search_html?q=
for Russian URL is
https://dict.dhamma.gift/ru/search_html?q=
or for compact mode use
https://dict.dhamma.gift/gd?search=

If you want to change the language just once, use Settings → Language inside the popup. Next time you click a word, it will open in your default language.
