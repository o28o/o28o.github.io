Охватываемые продукты

Сервисы Telegram

  • Телеграм боты:
    • @Dhammagift_bot - Основной бот для поиска по суттам и пали-словарю
    • @Dgift_bot - Короткое имя бота
  • Мини-приложения Telegram:
    • Мини-приложение поиска - Для поиска сутт внутри Telegram
    • Мини-приложение чтения - Для чтения пали-текстов
    • Мини-приложение словаря - Для поиска в словаре

Браузерные расширения

  • Расширение для Chrome
  • Расширение для Edge

Обрабатываемые данные

Эти продукты обрабатывают только минимальные технические данные, необходимые для их работы:

  • Для поиска в словаре: Передаётся только выбранное/кликнутое слово
  • Для поиска сутт: Передаются только поисковые запросы и ссылки на тексты
  • Технические данные: Базовая метаинформация запросов (IP-адрес, user agent), которую собирают все веб-серверы

Что мы НЕ собираем

  • Никакой личной информации пользователей
  • ID пользователей Telegram или истории чатов
  • Историю посещений вне наших сервисов
  • Постоянные идентификаторы для отслеживания

Передача данных

Все данные передаются только на наши сервисы:

  • dhamma.gift - Для поиска текстов и чтения
  • dict.dhamma.gift - Для поиска в словаре
  • dpdict.net - Для полных результатов пали-словаря

Данные пользователей не передаются третьим сторонам.

Хранение данных

Мы не храним:

  • Поисковые запросы или слова после обработки
  • Данные об индивидуальных паттернах использования
  • Историю взаимодействий с Telegram ботами

Логи серверов хранятся максимум 30 дней только для технических нужд.
